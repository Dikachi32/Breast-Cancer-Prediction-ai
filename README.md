# OncoPredict AI

An AI-powered breast cancer prediction system designed to support intelligent early-stage cancer screening using Machine Learning and Ensemble Learning techniques.

---

## Overview

OncoPredict AI is a healthcare-focused machine learning application developed to analyze clinical biomarkers and predict possible breast cancer diagnoses within seconds.

The system integrates multiple machine learning models, including:

- Logistic Regression
- Random Forest
- Gradient Boosting
- Ensemble Learning Model

The Ensemble Model combines predictions from multiple trained algorithms to improve:

- Accuracy
- Precision
- Stability
- Reliability

The platform provides:
- Instant diagnosis prediction
- Cancer risk percentage
- Severity awareness
- Intelligent healthcare support

> ⚠️ Disclaimer:
> This project is NOT designed to replace medical professionals.
> It is developed to assist healthcare experts through intelligent early-stage screening awareness and predictive support.

---

# Features

## AI Diagnosis System
- Predicts possible breast cancer diagnosis in approximately 2 seconds.
- Processes clinical biomarkers instantly.

## Multiple Machine Learning Models
Users can test predictions using:
- Logistic Regression
- Random Forest
- Gradient Boosting
- Ensemble Learning

## Ensemble Learning Recommendation
The Ensemble Model is the recommended prediction model because it combines outcomes from multiple trained models to improve overall prediction performance.

## Cancer Risk Analysis
The system provides:
- Risk percentage score
- Severity indication
- High-risk medical alert recommendations

## Real-Time Prediction
Fast and responsive prediction interface with immediate analysis feedback.

---

# Machine Learning Models Used

## Logistic Regression
Used for binary classification of benign and malignant cases.

## Random Forest
Provides decision-tree-based ensemble classification.

## Gradient Boosting
Improves prediction performance using sequential boosting techniques.

## Ensemble Learning
Combines predictions from:
- Logistic Regression
- Random Forest
- Gradient Boosting

This improves:
- Model stability
- Prediction confidence
- Generalization performance

# How the System Works

1. User enters clinical biomarker values.
2. User selects a machine learning model.
3. The AI system analyzes the biomarkers.
4. Prediction result is generated instantly.
5. The system displays:
   - Diagnosis result
   - Cancer risk percentage
   - Severity level
   - Recommendation alert

---

# Sample Prediction Workflow

Example:
- User enters biomarker data
- Ensemble Model analyzes the input
- System predicts:
  - Malignant
  - 75.03% High Risk

The platform then recommends immediate medical consultation.

# Why Ensemble Learning?

The Ensemble Learning Model combines predictions from multiple trained algorithms.

Benefits include:
- Higher accuracy
- Better reliability
- Reduced overfitting
- Improved consistency

This makes the Ensemble Model the most recommended option in the system.
